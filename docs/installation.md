> Prerequisite : mobx-react

NPM

```
npm install mobx-react
```

Yarn

```
yarn add mobx-react
```

You can install @usedb/core and @usedb/react as a package from NPM or yarn.

NPM

```
npm install @usedb/core @usedb/react
```

Yarn

```
yarn add @usedb/core @usedb/react
```
